# Installation

## From source

The source files for OSM Placenames can be downloaded from the [Github repo](https://github.com/rasher/osm_placenames).

You can either clone the public repository:

```sh
git clone git://github.com/rasher/osm_placenames
```

Or download the [tarball](https://github.com/rasher/osm_placenames/tarball/master):

```sh
curl -OJL https://github.com/rasher/osm_placenames/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd osm_placenames
uv pip install .
```
